Twitch Cat Overlay

This is a cat themed interactive overlay for Nosfear.
Users who talk in chat are roaming on the screen with their matching username's colors and roles.
Commands can be used to interact with the streamer's cats, to start a duel between two viewers or even to throw bombs at each others !

The application is running on the Godot Engine, the Twitch handling is custom made in C#.
Everything else is programmed in GDscript.
