Pathfinder Currency Converter

A tool to easily convert currency from the tabletop rpg game Pathfinder: Kingmaker.

I created this tool because the ones I found online were not adapted to this version of the game.

There are 2 versions of the tool:
● A simple conversion tool for additions or subtractions that I made quickly
● A wallet style tool that allows you to easily convert between currencies and add it into a global count (Javascript by my friend Nosfear, HTML/CSS by me)

I do prefer the second version, it is more user friendly and easy to use.
That's why developers should always ask other peoples for feedback because their ideas might be better than what we can come up with.
