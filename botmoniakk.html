BotMoniakk

BotMoniakk is a Python based Twitch moderator and assistant.
The bot integrates with the Twitch API and uses the TwitchIO library to interact with the Twitch chat.

Features:

● Analyzes each chat messages and assign a category that represents the message's content, this process is extremely fast and lightweight.

The categories are the following:
- Sexual
- Hate
- Violence
- Harassment
- Self-harm
- Sexual / Minors
- Hate / Threatening
- Violence / Graphic
- OK / Not offensive
The AI model used is this one, it is far from perfect but it works.

● Identifies and ban bots advertising scam websites. (it banned so far around a hundred users)

● Can respond to chat messages using the newly implemented LLM (Large Language Model) powered brain.
The AI model is currently running on my Ryzen 5 3600 (my AMD GPU is not compatible) and outputs responses in roughly 4 to 8 seconds depending on the output tokens length.
The LLM used is the lightweight but smart stablelm-zephyr-3b-GGUF model by TheBloke.

● Logs moderation actions in the Discord server using webhooks.

● Warns me of his actions using sound alerts.


The main goal of the bot was having an AI moderator doing the work of a human but the technology isn't accurate enough for now.
(also my AMD graphic card cannot be used for AI related stuff so I have to use my CPU which limits a lot my options)
