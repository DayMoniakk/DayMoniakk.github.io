Operation: Earthquake

Operation: Earthquake is a short top down shooter made for the LÖVE Jam 2022.
I wasn't really inspired by the theme so I've implemented it somehow into the game scenario.
My main objective was to create a small game that I will be able to finish and I think I succedeed !

This is my first ever project released using the Lua programming language and this framework.
