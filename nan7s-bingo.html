Everyday Nan7s Bingo

A minigame created for the Twitch streammer nan7s_peluche.
The board contains 25 randomly generated prompts that we expect the streamer to say, do, or mention.

The website is built using the Vue frontend framework.
It used to be made in Svelte 4 but Svelte 5 drove me crazy so I decided to switch, the transition process was fairly smooth.
